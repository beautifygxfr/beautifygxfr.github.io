(旧版本)

以下教程不需要深入理解 照着做就对了！

我们先来改Dex6

首先打开Arsc 复制 bg_texture_theme_version2 这个文件的ID (763的Id为 7F0202C7

然后再复制 aio_face_bg.9.png 这个透明图片的id 注意如果这张图不是透明的请把它改成透明图片尺寸随意需要点九 (763版本的Id为 7F020043)

不会复制ID的看这里

以Dex编辑器打开Dex6

十六进制搜索刚才复制的 Id 7F0202C7 ( bg_texture_theme_version2.xml)

搜索出来的结果如下 我们要改的就是红框内的

我们一个一个改 打开第一个 继续搜索 7F0202C7

会得到两个方法 (两个都要改)

先改第一个 打开以后 点击右上角

继续搜索这个Id 7f0202c7 (这里要注意 id要小写的 不然可能搜索不出来)

然后点击 替换 输入 7F020043 (aio_face_bg.9.png 这张透明图片的id )

如下图 最后点击全部这个按钮


最后返回 保存并退出

刚才改的只是第一个方法 还有一个方法也需要改

当然了红框内的都需要改 都一样改

刚才把主页消息 联系人 动态这三个界面的顶栏都透明了

下面我以Rz bdkd.png 为例子
