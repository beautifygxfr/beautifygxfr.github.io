防撤回

Dex3

Dex编辑打开dex3，常量搜 “对方”
打开 com/tencent/mobileqq/app/message/BaseMessageManager
继续搜索对方 得到方法b 打开后搜索fold redbag

往下数四行把const/4 v3 1 改成const/4 v3 0

Resource搜“撤回了一条消息”并复制其ID

然后搜“你常打开这个群，是否移出群助手”并复制其ID

Dex++编辑打开dex3，搜BaseMessageManager并打开BaseMessageManager

搜索“撤回了一条消息”的ID，并替换为“是否移出群助手”的ID（只替换第一处）

然后返回QQ包，在resource里修改“你常打开这个群，是否移出群助手”（对方的）为自定义的文字，而“撤回了一条消息”就是自己撤回消息的提示
